# DashBoards Futbol

Proyecto de business intelligence ejecutivo para clubes de fútbol argentinos.

## Qué es esto
Dashboards en Power BI para dirigencias de clubes (presidente, comisión directiva).
El ángulo es gestión institucional — no scouting, no táctica.

## Estructura
```
.claude/skills/club-dashboard-builder/  — skill para construir dashboards
.claude/rules/dashboard-rules.md        — reglas de datos y diseño
[club]/                                 — carpeta por club (ej: Velez/, Racing/)
```

## Cómo trabajar
Para construir un dashboard, leer primero `.claude/skills/club-dashboard-builder/SKILL.md`.
Las reglas en `.claude/rules/dashboard-rules.md` aplican siempre, sin excepción.
